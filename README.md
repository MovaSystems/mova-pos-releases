## 🚀 خطوات التثبيت

1. حمّل ملف `Mova POS Setup x.x.x.exe` من [صفحة الإصدارات](https://github.com/MovaSystems/mova-pos-releases/releases/latest).
2. شغّل الملف واتبع خطوات التثبيت.
3. عند أول تشغيل، سيُطلب منك مفتاح تفعيل — تواصل مع فريق الدعم للحصول عليه، أو ابدأ التجربة المجانية (١٥ يوم).
4. جاهز — يمكنك البدء في تسجيل عمليات البيع فوراً.
